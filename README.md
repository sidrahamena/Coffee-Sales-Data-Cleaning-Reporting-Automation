# ☕ Coffee Sales Data Cleaning & Automation

## 📌 Project Overview
This project focuses on automating the data cleaning and preprocessing workflow for a messy coffee sales dataset. The goal is to transform raw transactional data into a clean, structured format that is ready for analysis and reporting.

## ⚙️ Key Features
- Automated handling of missing values
- Removal of duplicates and inconsistent entries
- Standardization of data formats (dates, categories, etc.)
- Data preprocessing using Python (pandas)
- Export of cleaned dataset for reporting and visualization

## 🛠️ Tools & Technologies
- Python
- Pandas
- Excel (for dataset handling/verification)

## 📊 Outcome
- Clean and structured dataset ready for analysis
- Improved data quality and consistency
- Automated preprocessing workflow for efficiency
